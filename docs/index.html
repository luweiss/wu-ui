WuUi文档

用于uni-app的一些UI组件，需要vue3.

组件列表

  • wu-button

    按钮

  • wu-cell

    以列表的方式展示内容的组件，通常配合wu-cell-group使用

  • wu-cell-group

    配合wu-cell使用

  • wu-checker

    单选、多选、开关

  • wu-icon

    使用iconfont图标

  • wu-input

    输入框

  • wu-input-number

    数字输入框

  • wu-popup

    半屏弹出框，从页面底部弹出

  • wu-scroll-view

    滚动内容

  • wu-section

    一个简单的由标题和内容组成的组件

  • wu-swipe

    可以左滑显示隐藏内容的组件

  • wu-test

    This is an example of creating a reusable grid component and using it with external data.
